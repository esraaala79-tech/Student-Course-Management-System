Student Course Management System (C++)

 Overview
Console-based system to manage students and their courses using C++ (OOP + STL).
 Features
- Add / Remove Student  
- Search Student (ID / Name)  
- Display Students  
- Enroll Courses  
- Show Courses  
- Sort by GPA  
- Save & Load from file  
 Technologies
- C++
- OOP
- STL (vector, set)

```bash
g++ main.cpp Student.cpp Person.cpp -o system
./system
